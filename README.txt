Research project: Topology-enhanced Video Classification

baseline.py:

baseline.py includes a baseline video classification model that will be used to test against topology-enhanced models.

TF_extracted.py:

TF_extracted.py copies the baseline video classification model and extracts topological features from the data. This file will serve as a blueprint for topology-enhanced models.

feature_fusionV1.py:

feature_fusionV1.py is the first file used to test different ways of combining topological representations with the baseline 3D CNN.
