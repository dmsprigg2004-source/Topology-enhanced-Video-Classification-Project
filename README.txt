Research project: Topology-enhanced Video Classification

baseline_3D_CNN.py:

baseline_3D_CNN.py includes a baseline 3D CNN video classification model and also extracts topological features from the data.

ffV1_3D_CNN.py:

ffV1_3D_CNN.py is the first file used to test different ways of combining topological representations with the baseline 3D CNN for improved video classification.
